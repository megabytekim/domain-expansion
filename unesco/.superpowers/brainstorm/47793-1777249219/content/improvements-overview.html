혜초 지도 — 개선 영역 후보

지금 어떤 부분이 가장 아쉬우세요? 여러 개 골라도 돼요.

A

같은 위치 상품 탐색

카트만두·돌로미테 같은 인기 목적지에 마커가 겹쳐있어 하나만 보임. 클릭하면 해당 위치의 모든 상품 목록을 볼 수 있어야 함.

현재: 437개 마커 중 같은 좌표 겹침 많음

B

검색 & 상세 필터

목적지명 검색, 가격대 슬라이더, 기간 필터 추가. 지금은 카테고리 4개만 있음.

예: "네팔" 검색 → 해당 상품만 하이라이트

C

상품 상세 정보 강화

바텀시트에 이미지·제목·가격만 있음. 일정 요약(Day 1~3 주요 도시), 출발일 정보, 경유 도시 목록 추가.

현재: 이미지 + 제목 + 가격 + 링크

D

지도 인터랙션 개선

마커 hover 시 상품명 툴팁 표시, 마커 크기 차등화(상품수에 비례), 클릭한 마커 강조 효과.

현재: hover 없음, 모든 마커 동일 크기

E

모바일 최적화

바텀시트 스와이프 제스처, 필터바 접기/펼치기, 작은 화면에서 지도 조작 편의성 개선.

현재: 터치 기본 지원되나 UX 미최적화
