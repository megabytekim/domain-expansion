# 중국 황산 (黄山) 여행 플래닝

> 날짜: 미정 | 인원: 미정 | 예상 일정: 3박 4일 (황산 2박 + 홍촌 1박 or 당일)

---

## 개요

| 항목 | 내용 |
|------|------|
| 위치 | 중국 안후이성(安徽省) 황산시(黄山市) |
| 공항 | 황산 툰시 국제공항 (TXN, 屯溪机场) |
| 베이스캠프 | 탕커우(汤口) — 케이블카·버스 출발지, 숙소 밀집 |
| 비자 | 한국→중국 단기 비자면제 여부 출발 전 최신 확인 필수 (정책 변동 잦음) |
| 환율 | 1 CNY ≈ 190-200원 (기준 수시 변동) |
| 통화 | 위안(¥, CNY). **현금+알리페이 병행 필수** (카드 안 되는 곳 많음) |
| 언어 | 만다린 중국어. 산 위는 영어 거의 불통 |
| 최적 시기 | 봄(3-5월), 가을(9-11월) — 운해·단풍. 여름 비·안개, 겨울 빙설도 장관이나 위험 |

---

## 황산 지형 한눈에 보기

```
                      [북해 구역]
                      사자봉 ★ (일출 명소)
                      배운루
          [서해 대협곡] ─── 광명정 ─── [동해]
          시하이 케이블카   (최고 전망)
                      |
               [옥병 구역]
               연화봉 (최고봉 1864m)
               천도봉 (영험한 오르막)
               영객송 ★ (사진 명소)
                      |
            탕커우 (汤口) 베이스타운
            ├── 운곡 케이블카 (→ 북해)
            ├── 옥병 케이블카 (→ 옥병/남해)
            └── 태평 케이블카 (→ 북해, 경치↑)
```

## 케이블카 정보

| 케이블카 | 방향 | 편도 요금 | 소요 | 특징 |
|----------|------|-----------|------|------|
| **운곡(云谷)** | 탕커우 → 북해 | 약 ¥100 | 8분 | 가장 많이 이용. 대기 길수 있음 |
| **옥병(玉屏)** | 탕커우 → 옥병 | 약 ¥90 | 8분 | 영객송·연화봉·천도봉 코스 접근 |
| **태평(太平)** | 탕커우 → 북해 | 약 ¥85 | 20분 | **경치 최고**. 반대편 하산 시 추천 |
| **시하이(西海) 협곡** | 산 위 내부 | 약 ¥80 | 10분 | 서해 대협곡 내부 이동용 |

> **전략**: 올라갈 때 운곡 or 옥병 → 내려올 때 태평 (경치 보며 하산)

## 추천 일정 (3박 4일)

```
D1  인천 → 황산(TXN) 도착 → 탕커우 체크인 → 온천
D2  케이블카 탑승 → 산 위 하루 종일 트레킹 → 산 위 숙박 (★핵심)
D3  일출 감상 → 오전 트레킹 → 태평 케이블카 하산 → 탕커우
D4  홍촌(宏村) 당일치기 → 황산공항 → 인천
```

## D1 — 도착 + 온천

| 시간 | 일정 | 비고 |
|------|------|------|
| - | 황산 TXN 공항 도착 | 인천→황산 직항 or 상하이·항저우 경유 |
| - | 탕커우(汤口) 이동 | 공항에서 버스/택시 약 1시간 |
| - | 숙소 체크인 | 탕커우 내 호텔 (산 아래) |
| 저녁 | 탕커우 로컬 식당 | 훠궈, 황산 마오더우푸(毛豆腐) 체험 |
| 밤 | **온천(温泉)** | 탕커우 온천가. 황산 온천은 약알칼리성 탄산천 |

### 탕커우 맛집 후보

| 음식 | 설명 |
|------|------|
| **마오더우푸(毛豆腐)** | 안후이 명물. 발효 두부 숯불 구이. 현지인들 간식 |
| **황산 훠거우(火锅)** | 산 지역이라 버섯류·산채 풍부 |
| **훈예(臭鳜鱼)** | 발효 쏘가리 찜. 냄새 강하지만 안후이 대표 요리 |
| **산나물 볶음** | 봄 시즌엔 죽순·고사리 등 산채 일품 |

## D2 — 황산 트레킹 (산 위 숙박)

> ★ 황산의 핵심은 **산 위에서 1박**. 일출·운해·석양을 모두 보려면 필수.

| 시간 | 일정 | 비고 |
|------|------|------|
| 06:30 | 케이블카 탑승 (운곡 or 옥병) | 성수기 대기 1-2시간 가능. 일찍 출발 |
| 오전 | **서해 대협곡(西海大峡谷)** 루프 | 협곡 뷰 최고. 내부 케이블카 이용 가능 |
| | **광명정(光明顶)** | 360도 파노라마. 황산 두 번째 봉우리 (1860m) |
| 점심 | 산 위 식당 | 비싸고 맛없지만 어쩔 수 없음. 간식·컵라면 준비 추천 |
| 오후 | **비래석(飞来石)** | 거대한 돌이 벼랑 끝에 떠 있는 모양 |
| | **몽필생화(梦笔生花)** | 소나무 한 그루 + 봉우리. 황산 대표 포토스팟 |
| 저녁 | **배운루(排云亭)** | 서해 일몰·운해 감상 명소 |
| 숙박 | **산 위 호텔** | 아래 참조. **최소 2-3개월 전 예약 필수** |

### 산 위 숙소 (예약 선착순, 성수기 매우 빠르게 마감)

| 호텔 | 위치 | 특징 | 요금 (참고) |
|------|------|------|-------------|
| **시하이 호텔(西海饭店)** | 서해 구역 | **위치 최고**. 서해 협곡 바로 앞 | ¥1,200-2,000/박 |
| **베이하이 호텔(北海宾馆)** | 북해 구역 | 가장 오래된 산 위 호텔. 사자봉 가까움 | ¥1,000-1,500/박 |
| **바이윈 호텔(白云宾馆)** | 광명정 근처 | 중간 위치, 동선 유리 | ¥900-1,400/박 |

> 예약: **씨트립(Ctrip/Trip.com)** 앱에서 중국 내 결제 가능. 구글에서 "Huangshan mountain hotel"

## 항공편

| 루트 | 항공사 | 소요 | 특이사항 |
|------|--------|------|----------|
| **인천 → 황산(TXN) 직항** | 에어서울, 중국동방항공 등 | 약 2시간 | 노선 편수 적음. 스케줄 확인 필요 |
| **인천 → 상하이(PVG) → 황산** | 다양 | 4-5시간 | 경유 많음. 기차 연결도 가능 |
| **인천 → 항저우(HGH) → 황산** | 다양 | 3-4시간 | 항저우에서 버스 약 3시간 |

> 직항이 있는 날짜면 TXN 직항 강추. 없으면 상하이 경유가 가장 무난.

## 트레킹 코스 난이도

| 구간 | 거리 | 난이도 | 소요 |
|------|------|--------|------|
| 운곡 ~ 북해 (케이블카 없이 도보) | 약 7.5km | ★★★★☆ | 3-4시간 |
| 서해 대협곡 루프 | 약 6km | ★★★☆☆ | 2-3시간 |
| 광명정 ~ 연화봉 | 약 3km | ★★★☆☆ | 1.5시간 |

> 전체 황산 트레킹 하루 평균 10-15km. 무릎 보호대 있으면 하산 시 유용.
