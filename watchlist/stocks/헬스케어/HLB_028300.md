## 6. 전략적 분석

### Sector Position
- **K-바이오 대표 기업**: 국산 항암제 FDA 승인 도전 중
- **직접 상업화 모델**: 기술수출 로열티가 아닌 직접 판매 추구
- **복합 파이프라인**: 간암 + 담관암 + CAR-T (다각화)

### Moat Assessment
| 요소 | 평가 | 근거 |
|------|------|------|
| Network Effects | No | 해당 없음 |
| Switching Costs | Low | 처방약 시장 |
| Cost Advantages | No | R&D 집중 단계 |
| Intangible Assets | **Medium** | 파이프라인 특허, 임상 데이터 |
| Efficient Scale | No | 경쟁 시장 |

### Investment Thesis

**Bull Case (+50% to 100,000원)**:
- FDA C1 분류 + 최종 승인 시 글로벌 간암 1차 치료제 시장 진입
- 담관암 라이라푸그라티닙 Best-in-class로 추가 파이프라인 가치
- 김태한 회장 리더십으로 CMC 문제 완전 해결
- 직접 판매 모델 성공 시 높은 수익성

**Bear Case (-50% to 35,000원)**:
- FDA 3차 CRL (불승인) 시 주가 급락 불가피
- CMC 문제 재발 또는 추가 요구사항
- 경쟁사 대비 시장 진입 지연으로 점유율 확보 실패
- 현재 과매수 상태에서 기술적 조정

**Base Case (±15%)**:
- FDA C2 분류 (6개월 심사)로 결과 지연
- 단기 조정 후 FDA 결과 대기 상태 유지
- 55,000~75,000원 박스권 형성
