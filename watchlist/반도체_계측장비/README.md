# 반도체_계측장비

## 개요
반도체 미세화/적층화로 패터닝 반복 및 레이어 수가 증가하면서 오버레이 계측장비 수요가 구조적으로 증가. KLA가 과점하는 시장에서 국내 업체(오로스테크)가 IBO 방식으로 침투 중. 후공정(하이브리드 본딩, TSV) 확장이 새로운 성장 축.

## 원본
자연어 입력 - 반도체 계측장비 산업 노트 (Thin Film, IBO/DBO, IR 오버레이, 후공정 라인업)
