# 항체설계AI 관련주 분석

> 분석일: 2026-03-16

## 테마 배경

서울대 윤태영·백민경 교수의 "그랜드퀘스트" 발표에서 출발. 알파폴드/로제타폴드 이후 차세대 난제인 **항체 설계 AI**와 **적응 면역계 이해 AI** 개발이 본격화되고 있다. 한국은 제조업 강점 기반 고품질 실험 데이터 구축 전략으로, 5년 내 항체 신약 설계 능력 확보를 목표로 한다.

**강의 요약**: [그랜드퀘스트 시즌 1 강의요약](강의요약_그랜드퀘스트_항체설계AI.md)
