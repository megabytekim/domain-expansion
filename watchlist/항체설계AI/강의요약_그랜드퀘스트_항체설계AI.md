# 그랜드퀘스트 시즌 1: 항체 설계 AI

> 단백질 구조 예측 AI를 넘어, 항체를 설계하고 생명체의 적응 면역계를 이해하는 AI를 만들 수 있을까?

**발표자**: 서울대학교 생명과학부 윤태영 교수, 백민경 교수
**원본**: [YouTube](https://youtu.be/CXTasbFkUzM?si=3N24kj_XCpLqkjFY)

---

## TL;DR

- 알파폴드/로제타폴드가 단백질 구조 예측은 풀었지만, **항원-항체 결합 예측 정확도는 0-28%**로 실패
- 원인: 항체 정보는 유전되지 않아 진화 데이터가 없고, 결합 부위(루프 영역)가 가변적
- 해법: **대규모 실험 데이터를 직접 구축**하여 AI가 항체 설계 원리를 학습하도록 접근
- 한국 전략: 제조업 강점 기반 고품질 데이터 구축 (주당 250→1,000 데이터포인트 목표)
- 목표: 10만→100만 데이터로 범용 항체 설계 AI 개발, **5년 내 성패 결정**

## 1. 단백질 구조 예측 AI의 성과

### 왜 단백질 구조가 중요한가

- 단백질은 수송, 효소, 신호전달, 면역(항체) 등 생명 현상의 핵심 분자
- 구조를 알면 생명 현상을 분자 수준에서 이해 가능
- 코로나19 사례: 스파이크 단백질-ACE2 결합 구조 이해 → 항체 치료제, 백신, 진단키트 개발

### 핵심 아이디어: 진화 정보 활용

- 진화적으로 연관된 단백질들을 정렬하면 **구조 관련 패턴**이 숨어 있음
- AI가 이 패턴을 찾아 3차원 구조를 예측 → 알파폴드, 로제타폴드

### 현재 성과

- CASP 대회에서 **90점 가까운 정확도** 달성
- 단일 단백질뿐 아니라 단백질 간 **상호작용 예측**도 가능
- 50년 넘게 해결 못한 난제를 해결한 AI 혁신

## 2. 항체 설계 AI가 어려운 이유

### 항체 신약의 구조

- 항체는 Y자 모양, 대부분 고정 영역(템플릿 리전)
- 끝부분 **CDR(Complementarity Determining Region)**: -60개 아미노산, 이 서열을 바꿔 다양한 항원에 결합
- 면역항암제 키트루다 등 **수십조 원 규모** 산업

### 기존 AI가 실패한 근본 원인

| 원인 | 설명 |
|------|------|
| **진화 정보 부재** | 항체 정보는 유전되지 않음 (획득 면역 → 생식세포 비전달). NGS 데이터에 항체 정보가 비어있어 AI가 학습 불가 |
| **학습 영역 불일치** | 기존 AI는 단백질 몸체(고정 부분) 예측에 특화. 항체 결합은 구조가 가변적인 **루프 영역** 중심 → 예측 취약 |
| **데이터 부족** | 항원-항체 결합 구조 데이터 자체가 부족 |

### 결과

- 알파폴드2만 사용 시 정확도 **0%**
- 특화 딥러닝 적용해도 **25-28%** → 신뢰 불가

## 3. 새로운 전략: 데이터 구축 + AI 학습

### 접근 방식 (지도 학습 기반)

```
CDR 서열 변형 → 항체 제작 → 항원 결합 점수 측정 → 데이터 축적 → AI 학습
                                                          ↑
                                                    AI 피드백으로
                                                    스마트 변형 설계
```

### 글로벌 선도 기업 비교

| 기업 | 방식 | 장점 | 한계 |
|------|------|------|------|
| **Absci** | DNA 라이브러리 + FACS + NGS | 한 번에 수천 개 측정 | 온/오프만 판별, 낮은 해상도 (82,000 포인트지만 로우 레졸루션) |
| **Generate Biomedicines** | 직접 항체 제작 + 구조 생물학 | 정답지(구조) 직접 생산 | 속도↔정확도 상충, 주 100개도 어려움 (암젠 1.9조 투자) |
| **윤태영 교수팀 (한국)** | 자체 장비·칩 제작 + 개별 이벤트 분석 | 해리 상수 직접 측정 (0-100점), 고해상도 | 현재 주당 250 → 목표 주당 1,000 데이터포인트 |

### 스마트 데이터 확장 전략 (백민경 교수)

1. PDB 내 -1,000개 기존 항원-항체 결합 구조를 출발점으로
2. 결합력 좋은 항체 주변 아미노산을 하나씩 변형 (산봉우리 주변 탐색)
3. 아미노산 개수 증감으로 탐색 범위 확장
4. AI 학습 → AI 설계 항체 실험 검증 → 피드백 → 반복
5. 목표: 효율적으로 **100만 데이터포인트** 구축

## 4. 항체 설계 AI의 파급 효과

### B세포 (항체) 측면

| 효과 | 설명 |
|------|------|
| 효율적 의약품 개발 | 100개 설계 중 1개 → **90개 성공** 수준으로 향상 |
| 개인 맞춤 의약품 | 프리시전 메디슨: 개인 특성에 맞는 항체 설계 |
| 이중 항체 | 타겟 2개 동시 결합, 멀리 있는 세포/물질 끌어오기 |
| 안전한 항암제 | 면역 체계 활용으로 부작용 감소 |

### T세포 (TCR) 측면

- 항체 설계 AI → TCR 설계로 확장 가능
- T세포 활용 **세포 치료제** 개발 (암세포 공격 T세포 주입 등)
- 면역학의 핵심 미스터리 (안테나 인지 메커니즘) 해결 열쇠

## 5. Q&A 핵심 정리

| 주제 | 핵심 내용 |
|------|----------|
| **AI의 논리 이해** | 신뢰할 수 있는 AI 연구 진행 중. AI가 찾은 패턴에서 사람이 놓친 물리 원리 발견 가능 |
| **데이터 문제** | 성공/실패 데이터 50:50 비율 필요. 실패 데이터 공유 문화 부재가 한계 |
| **데이터 해상도** | 윤 교수팀: 해리 상수 직접 측정 가능 (FACS의 온/오프 대비 월등), 클로닝 생략으로 속도 확보 |
| **알고리즘** | 밑바닥 개발보다 LLM/알파폴드 기반 알고리즘을 항체 특화로 변형 |
| **양자 컴퓨터** | 조합 최적화에 도움 가능, 현재 적용은 미지수 |
| **mRNA 백신** | 개인별 면역 반응 차이 예측 불가 → AI 면역계 이해 필요 |
| **범용 AGI** | 생명 현상 전체 이해 AGI는 40-100년 이상 소요 예상 |

## 6. 투자 시사점

- **시간 프레임**: 5년 내 항체 설계 AI 성패 결정 → 2026-2030 핵심 모니터링 구간
- **한국의 포지션**: 생산(CDMO) 강점 + 자체 데이터 구축 장비 → 설계 능력 확보 시도
- **수혜 체인**: 항체 CDMO → 항체 신약 설계 → 이중항체/ADC → 세포치료제(TCR)
- **리스크**: 데이터 구축 속도, 글로벌 경쟁(Absci, Generate Bio), 정책 투자 의지
